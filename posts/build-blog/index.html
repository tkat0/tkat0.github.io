再びblogを

Feb 1, 2019 10:00 · 1 minute read hugo

GitHubを整理していたら、何年か前にはてなブログからhugo@github.ioへ移行しようとした残骸を見つけたので復活させた。

エンジニアブログ書くならmediumとかnotesとかqiitaどれが良いんじゃろと数分考えて、以下を満たすのがhugoだった。 多分同じことを前にも考えてやろうとしていたのだと思う。

  • markdownでgit管理しながら書きたい
  • 好きなエディタでローカルで作業したい
  • codeのsyntax highlight
  • 日々の雑記だけでなく、まとめ記事も作成したい

syntax highlightの例。いいねー。

import tvm
print(tvm.__version__)

$ ls

std::cout << "hello world" << std::endl;

普段も調べたことはEvernoteに全部書いているけど、せっかくだしpublicにしたい。publicにすることで、 多少は読みやすく書くだろうし、後々自分にとっても良いだろうということ。 以下の通り、書いて忘れるのは大事だなと思った。最近色々手を伸ばしすぎて頭が整理できていないのでなお。

私は忘却力が高くて20代の頃からこんな感じでメモを書いては忘れて脳内メモリ空けてました。最近は検索できて良いですよね。 https://t.co/Uyb9Tp51oi
