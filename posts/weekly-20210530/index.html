振り返り

続けたいこと

  • Google IO 2021 を見ていて、Flutter が面白そうなので余暇で入門している
    • 公式の Youtube 動画がわかりやすい。英語の勉強にもなる。
    • 趣味でやりたいことはだいたいブラウザで完結しそうなので、今まで React などを勉強してきたが、Flutter Web はどうなんだろう？
    • まだ入門したてだけど、低コストできれいな GUI をつくるのは Flutter の方が簡単な印象
      • 既存の Widget の組み合わせで、あまり CSS のようなものをいじらなくても簡単にかけるように見える

改善したいこと

  • 先週、 散歩でもして気分を切り替えるのが良いかもしれない と書いたけど結局机の前で悩む時間が多かった
    • 今週は、検証の仕方が問題があった。問題を検証する最低限の環境を用意してやるべきが、本番システム相当のコードベースの中で検証をしてしまったので、検証が遅いし原因の切り分けは時間はかかるし、で良くなかった。
    • 仕事中にリフレッシュする良い方法はないだろうか。
