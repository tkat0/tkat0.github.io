Raspberry Pi 3 B+を買ったので初期設定のメモ

  • 立ち上げ
    • 直接Raspbianを焼いた。Etcherという便利なツールがあり、ddで焼かなくてよかった
    • キーボードとディスプレイを接続し、起動
    • WiFi設定などもここ
    • sudo raspi-config
    • sshの有効化
    • IPアドレス固定のため、 /etc/dhcpcd.conf を設定
    • reboot
  • sshで作業
    • Dockerのインストール
      • https://docs.docker.com/install/linux/docker-ce/debian/
    • sudo apt install vim
