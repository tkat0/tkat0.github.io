Raspberry Pi 3 B+買ったよ

Raspberry Pi 3 B+を買ったので初期設定のメモ

  • 立ち上げ
    • 直接 Raspbian を焼いた。Etcher という便利なツールがあり、dd で焼かなくてよかった
    • キーボードとディスプレイを接続し、起動
    • WiFi 設定などもここ
    • sudo raspi-config
    • ssh の有効化
    • IP アドレス固定のため、 /etc/dhcpcd.conf を設定
    • reboot
  • ssh で作業
    • Docker のインストール
      • https://docs.docker.com/install/linux/docker-ce/debian/
    • sudo apt install vim
