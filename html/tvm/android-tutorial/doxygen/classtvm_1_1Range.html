Detailed Description

same as HalideIR::IR::Range except it provide an constructor with (begin, end)

Note
Traditional Halide's Range have a constructor with (begin, extent), which does not match the convention in e.g. python. We decided to correct it by removing the constructor in HalideIR, and add it back in TVM's range.
