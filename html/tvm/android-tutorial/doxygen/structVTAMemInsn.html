Detailed Description

VTA load/store instruction Load/store instruction can describe a 2D strided access pattern with padding, which can be useful to perform spatial padding on the fly on a tensor on which to perform 2D convolution. For instance if we try to load a 4x4 spatial tile from a 16x16 matrix with padding of size 1 on all dimensions: y_size = 4, x_size = 4, x_stride = 16, y_pad_0 = 1, y_pad_1 = 1, x_pad_0 = 1, x_pad_1 = 1.
