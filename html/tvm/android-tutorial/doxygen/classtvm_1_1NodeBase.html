Detailed Description

Base class of Node for runtime destructor purposes.

Node is a reference counted object which is used to construct AST. Each node is backed by a custom deleter, which deletes the object. Do not call create raw Node pointer, always use tvm::make_node.

Note
In most cases, please inheritate tvm::Node.
See also
Node, NodePtr, make_node
