Detailed Description

Registry entry for NodeFactory.

There are two types of Nodes that can be serialized. The normal node requires a registration a creator function that constructs an empty Node of the corresponding type.

The global singleton(e.g. global operator) where only global_key need to be serialized, in this case, FGlobalKey need to be defined.
