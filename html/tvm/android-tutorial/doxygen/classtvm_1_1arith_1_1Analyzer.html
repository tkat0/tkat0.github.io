Detailed Description

Analyzer that contains bunch of sub-analyzers.

Each sub-analyzer can make use of another sub-analyzer by weak reference of this.

NOTE for sub-analyzer developers: If the analyzer uses memoization, we need to clear the internal cache when information about a Var has been overrideen.
