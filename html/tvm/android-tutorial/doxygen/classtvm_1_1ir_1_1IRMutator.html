Detailed Description

a base class for mutator to iterative mutate the IR

This IRMutator is implemented via Visitor Pattern. Also you can implement via IRFunctor. This enables easy extensions of possible new Node. It also makes changing return types easier.

Note
If you want to return a different type other than Expr and Stmt, Simply following the same pattern as IRMutator and create a seperate class.
See also
IRFunctor
