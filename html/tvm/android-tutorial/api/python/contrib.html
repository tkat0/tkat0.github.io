tvm.contrib.random.uniform(low, high, size)Â¶

Draw samples from a uniform distribution.

Samples are uniformly distributed over the half-open interval [low, high) (includes low, but excludes high). In other words, any value within the given interval is equally likely to be drawn by uniform.

Parameters:
  • low (float) â Lower boundary of the output interval. All values generated will be greater than or equal to low.
  • high (float) â Upper boundary of the output interval. All values generated will be less than high.
  • size (tuple of ints) â Output shape. If the given shape is, e.g., (m, n, k), then m * n * k samples are drawn.
Returns:

out â A tensor with specified size and dtype.
Return type:

Tensor
