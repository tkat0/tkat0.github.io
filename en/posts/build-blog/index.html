GitHub を整理していたら、何年か前にはてなブログから hugo@github.io へ移行しようとした残骸を見つけたので復活させた。

エンジニアブログ書くなら medium とか notes とか qiita どれが良いんじゃろと数分考えて、以下を満たすのが hugo だった。 多分同じことを前にも考えてやろうとしていたのだと思う。

  • markdown で git 管理しながら書きたい
  • 好きなエディタでローカルで作業したい
  • code の syntax highlight
  • 日々の雑記だけでなく、まとめ記事も作成したい

syntax highlight の例。いいねー。

print(tvm.__version__)
